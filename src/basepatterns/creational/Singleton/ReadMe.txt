Паттерн Singeton (одиночка):
Цель:
Для создания единтсвенного экземпляра класса
Применение:
-Необходим только один экземпляр конкретного класса, который доступен для всех клиентов
-Единственный экземпляр должен наследоваться путём порождения подклассов, при этом, клиенты имеют возможность работать с классом-наследником без модификации своего кода
